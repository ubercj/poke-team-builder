# Pokemon Team Builder

![Team Builder with Pikachu in search](/src/sample-pic.png)

[See it live.](https://ubercj.github.io/poke-team-builder/)

A demo of [React Query](https://react-query.tanstack.com/) and [Tailwind Css](https://tailwindcss.com/) for Thoughtworks Tech Radar 2021. A simple app that lets you search and construct a team of Pokemon, using information from [Pok√©API](https://pokeapi.co/).
